A general pattern of a CLI:

Prompt         command         param 1 param 2 param 3 ..... Param n

**Prompt** - can give you the information. 

Generated by the sys/program to provide context for the user.

**Command** - Provided by the user.
3 types of command:
1. Internal command - inbuilt command in the system.
   It is a command that exists within a system.
   Which ku response vantha internal... varalana included.
2. Included - run separate executables.
3. External - executables that run on command and has / access 3rd party libraries (Apps).
